# Data Cleaner App

Starter Flask project for uploading and cleaning CSV and Excel files.

## Setup

1. Create a virtual environment.
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run the app:
   `python run.py`

## Current Status

This first chunk includes:
- app factory setup
- SQLite database configuration
- Flask-Login setup
- user model
- guest user utility
- Bootstrap-based starter pages

The next chunk can add file upload, preview, cleaning, and download features.
